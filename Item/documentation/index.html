Using the icons

IonIcons

In this template I've included the ionicons icon pack, to not have difficulty to you using them, I have wrote this to make it easier for you to use the icons. It is just simple. Follow the steps:
The icon's markup is:

<i class='icon-class-name' >

To find the class name of the icon, go to the ionicons.com. There you will find the icons on the first page, then just click on the icon you want to use and the popup will be appeared, on that popup, the class name of the icon will be written, just add that classname to the <i> tag, and you will get the icon.

LineaIcons

To use linea icons, you need to open the folder "linea_iconfont" and there you'll find a html file, which contains the list of all icons. There, is CSS MAPPING section, you will find the names of icon under them. To apply icon, just add "icon-X" class to and <i> element where X is the name of icon.
For Example: <i class='icon-basic-accelerator' ><i>
